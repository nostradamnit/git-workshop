git commands for the beginners

  • git init initialize a repository
  • git status show the status
  • git add * add all untracked files
  • git commit -m "<awesome message>" commit changes with awesome message
  • git remote add origin <:URL remote repo> add remote repository
  • git push push changes to remote repository
  • git clone clone a remote repository locally
